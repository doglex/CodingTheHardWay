# 金融市场
+ 基本资产：股票、证券、货币(人民币换美元)
+ 金融衍生产品：远期合约、期权、期货等
+ 多头：表示买入
+ 空头：没有资产却要卖出。比如先借了股票，之后还相同数量的股票。我国有限制
+ 套利：没有风险的获取利益机会。成熟的市场是很少的，瞬间就没了
